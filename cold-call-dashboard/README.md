# Дашборд холодных обзвонов

Встроенный дашборд для Битрикс24 для контроля процесса холодных обзвонов.

## Возможности

- **Срочно позвонить** — список клиентов на стадии "Холодный обзвон", отсортированных по давности последнего звонка
- **Звонки за сегодня** — количество завершённых звонков
- **Конверсия** — процент сделок, перешедших из холодного обзвона в следующую стадию
- **Быстрый звонок** — кнопка для мгновенного звонка клиенту через телефонию Битрикс24
- **Автообновление** — данные обновляются каждые 60 секунд

## Установка

1. Загрузите файлы на сервер или в Битрикс24
2. Убедитесь, что доступен BX24 JS SDK
3. Настройте название стадии в `js/api.js` (переменная `coldCallStageName`)

## Настройка

В файле `js/api.js` измените:
```javascript
config: {
    coldCallStageName: 'Холодный обзвон', // Название вашей стадии
}
```

## Требования

- Битрикс24 с включённой CRM
- Телефония Битрикс24 (для кнопки звонка)
- Права на чтение сделок и активностей
